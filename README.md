						      CANTINA SIMULATOR 2026 	
————————————————————————————————————————————————————————————————————————

1. Nombre del proyecto: Cantina Simulator 2026
2. Integrantes: Luca Sprovieri, Theo Gonzalez
3. Descripción: es un juego 2D de gestión y simulación social donde administrás el buffet del colegio. Deberás controlar el stock, atender pedidos a contrarreloj y dialogar con alumnos y docentes. Tomá decisiones estratégicas para mantener la economía a flote y evitá acumular tres infracciones normativas que provoquen el cierre definitivo de tu negocio.
4. Tecnologias principales: Lenguaje: Java 21 LTS, Framework: LibGDX 1.14.2, Plataforma Objetivo: Escritorio (Desktop - LWJGL3)

6. Instrucciones Básicas de Compilación y Ejecución:
		PASO 1: Clonar este repositorio.,
		PASO 2: Abrir el proyecto en IntelliJ IDEA.,
		PASO 3: Esperar a que Gradle termine de sincronizar todas las librerías.,
		PASO 4: Buscar la clase Lwjgl3Launcher dentro de lwjgl3/src/main/java/ y darle al botón de Run (Play).

(Nota: Cambiá donde dice "Tu Nombre Completo" por tus datos, y poné tu usuario de GitHub en el link de la Wiki).
