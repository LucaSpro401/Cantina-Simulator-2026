# Cantina-Simulator-2026
Cantina Simulator 2026 es un videojuego 2D de gestión, simulación económica y desarrollo de diálogos desarrollado en Java con libGDX. El jugador administra la cantina de una escuela atendiendo clientes, preparando pedidos, controlando recursos y gestionando el progreso de la partida mediante una base de datos SQLite con JDBC.
